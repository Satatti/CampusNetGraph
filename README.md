# CampusNetGraph

Proyecto desarrollado en C# utilizando arquitectura MVC y grafos dirigidos.

## Integrante

- Santiago Valencia

## Responsabilidades

- Implementación del grafo dirigido
- Implementación BFS y DFS
- CRUD de usuarios y relaciones
- Pruebas y validaciones

## Funcionalidades

- Grafo dirigido con lista de adyacencia
- BFS desde diferentes usuarios
- DFS completo
- Detección de ciclos
- Consultas sociales
- CRUD de usuarios
- CRUD de relaciones
- Alcanzabilidad entre usuarios

## Tecnologías

- C#
- .NET Console App
- Arquitectura MVC

## Ejecución

1. Abrir la solución en Visual Studio
2. Ejecutar con CTRL + F5

## Evidencias

El programa imprime:
- Lista de adyacencia
- BFS
- DFS
- Detección de ciclos
- CRUD completo
